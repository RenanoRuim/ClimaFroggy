# 🌦️ ClimaFroggy

O **ClimaFroggy** é uma aplicação web desenvolvida para oferecer informações meteorológicas de forma simples, rápida e intuitiva. O projeto permite que os usuários consultem dados climáticos de diferentes localidades, apresentando informações como temperatura, condições do tempo e outros dados relevantes para facilitar o acompanhamento da previsão.

A aplicação foi construída utilizando uma arquitetura **Full Stack**, separando o frontend e o backend para garantir uma melhor organização, escalabilidade e manutenção do código.

No frontend, foi utilizado **React com Vite**, criando uma interface dinâmica, responsiva e agradável para o usuário. O backend foi desenvolvido com **Node.js e Express**, sendo responsável pelo gerenciamento das requisições e integração com serviços externos de dados climáticos.

## 🚀 Funcionalidades

* 🔎 Pesquisa de cidades para consulta do clima
* 🌡️ Exibição de temperatura e condições meteorológicas
* ☁️ Interface visual dinâmica baseada nos dados recebidos
* 📱 Design responsivo para diferentes dispositivos
* ⚡ Comunicação entre frontend e backend através de API

## 🛠️ Tecnologias utilizadas

### Frontend

* React
* Vite
* JavaScript
* CSS
* Axios

### Backend

* Node.js
* Express
* APIs externas

## 🎯 Objetivo do projeto

O ClimaFroggy foi desenvolvido com o objetivo de aplicar conceitos de desenvolvimento Full Stack, trabalhando integração entre interfaces, servidores, APIs e organização de projetos modernos.

Além de ser uma aplicação funcional, o projeto representa uma prática de desenvolvimento de software, explorando conceitos importantes como consumo de APIs, gerenciamento de dados, arquitetura de aplicações e deploy em ambientes de produção.
